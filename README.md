## 前言
  `Work_display`项目目前是一个专门用来展示过往作品的仓库。

## 项目介绍
    基于GitHub的Page服务的图片展示平台，专注提供优质展示服务，努力为个人作品宣传打造开发解决方案。
    
### 组织结构
``` lua
Work_display
├── 基于深度学习的微博话题检测系统 
├── 智录生分app-基于YXing与Android5xlsx框架的学生成绩管理及分析系统
├── 基于springboot的Coser展示平台
├── XJMH静态视频新闻网站
├── 基于easyX的层叠消融游戏

```
